# V2ray 管理脚本

因图形界面客户端难以为所有连接自定义配置, 所以编写此脚本

仅支持 Linux

# 使用方法

将 v2ray 配置文件置于此脚本所在目录, 并重命名为 `V2ray-Config.json`. 运行此脚本

可选地, 向 PATH 中添加此脚本的软链接

```sh
ln -s ./V2ray-Manager.py ~/.local/bin/v2man
```
